Overview

 

SQL Nexus is an extensible diagnostics collection, reporting, charting, and analysis tool.  It automates both the collection and analysis of diagnostic data, particularly that related to SQL Server.

 

SQL Nexus uses the SQL Server database engine to collect and aggregate diagnostic information.  In its current incarnation, this data relates to SQL Server itself, particularly the performance of SQL Server.  A primary use of this version of SQL Nexus is to diagnose and troubleshoot SQL Server-related performance issues.

 

SQL Nexus uses SQL Server Reporting Services technology to generate reports and charts based on the diagnostic data it collects.  This means that you can customize the reports and charts that ship with SQL Nexus or create your own using both SQL Server Reporting Services and Visual Studio.

 

SQL Nexus uses SQL Server’s SQLDiag facility to collect real-time diagnostic data that it then logs in a database for easy aggregation and analysis.  SQLDiag is an industrial-strength, high-performance diagnostic collector that specializes in SQL Server-related data collection.
