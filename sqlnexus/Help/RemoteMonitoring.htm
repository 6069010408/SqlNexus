Remote Monitoring

 

You can run SQL Nexus on a remote machine, but the data collector always runs on the server being monitored.  Before you can perform remote monitoring, you must run SQL Nexus once on the server itself and answer Yes when SQL Nexus asks whether you want to start the data collector; this will register the data collector as a service.  You cannot perform this initial service registration remotely.  However, once the data collector service has been registered, you can monitor and start/stop data collection by running SQL Nexus from any remote machine.
