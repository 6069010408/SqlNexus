Emailing Reports

 

SQL Nexus provides built-in support for creating emails containing analysis reports as attachments.  To create a new email and attach the current report to it as an attachment, simply click the Email button on the Report toolbar.  You can choose to attach the current report or all open reports.

 

SQL Nexus renders reports email reports as Excel worksheet files (XLS).  If you wish to attach a report in a format other than XLS, simply export it first, then manually create a new email and attach the result file.

 

You can customize the email template that SQL Nexus uses to create new emails by editing the Analysis.doc file in the Docs subfolder using Microsoft Word.  SQL Nexus will always start each new email with this file, so editing it will affect all new emails.
